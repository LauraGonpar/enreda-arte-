# 🚀 Hito 4: Integración y Despliegue - EnredaArte 

## 🔗 Enlace de Producción 
Link de la Aplicación Cliente (Producción): https://enreda-arte-1.onrender.com/

Este repositorio contiene la entrega final correspondiente al Hito 4, enfocado en la integración total y el despliegue de la aplicación a internet. El proyecto ha sido migrado de un entorno local a una arquitectura en la nube, garantizando la disponibilidad pública y la persistencia de datos.
## 📋 Descripción del Proyecto
En esta etapa, se consolidó la unión entre el cliente (Frontend) y el servidor (Backend), permitiendo que la aplicación gestione información real a través de una base de datos distribuida. Se han utilizado plataformas de servicios en la nube para asegurar un entorno de producción estable.
## 🛠️ Tecnologías y Despliegue
Para cumplir con los requerimientos de este hito, se utilizaron las siguientes herramientas:
Frontend: React, desplegado de forma independiente.
Backend: Node.js y Express, desplegado en la plataforma Render.
Base de Datos: PostgreSQL, con despliegue en la nube para asegurar la gestión remota de datos.
